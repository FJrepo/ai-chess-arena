# Logo and Icon Licensing

This repository does not ship official provider logos.

All files in `frontend/public/provider-logos/` are custom, in-house abstract icons created for this project to avoid third-party trademark usage risk.

## Policy
- Official company logos are excluded unless written permission is documented.
- Provider names in text are used for nominative reference only.
- Unknown providers fall back to a generic icon in the UI.

## If You Want to Add Official Logos Later
1. Obtain written permission from the trademark owner.
2. Save approval artifacts in `docs/logo-approvals/`.
3. Update this file with approval date, source URL, and scope.
4. Keep a custom fallback icon for each provider.
